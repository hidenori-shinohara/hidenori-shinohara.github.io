# Proposition
Find the fixed points in $\hat{\mathbb{C}}$ of $f(z) = \frac{z^2 - 1}{2z + 1}$.

# Solution
$z = \infty$ is clearly a fixed point.
Other than that, by solving $f(z) = z$, we obtain $z = (-1 + \sqrt{3}i) / 2$.
